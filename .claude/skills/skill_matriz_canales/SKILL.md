# skill_matriz_canales

## Estado
activa_documental

## Objetivo
Elegir dónde invertir esfuerzo y presupuesto para captar clientes.

## Cuándo usarla
Usar cuando la estrategia base (cliente y mensaje) esté clara.

## Agente activador
`estratega_marketing`

## Entradas necesarias
- `03_cliente_objetivo_y_segmentos.md`
- `04_propuesta_valor_y_posicionamiento.md`
- Tipo de servicio (B2B/B2C, ticket alto/bajo).

## Proceso
1. Evaluar ciclo de venta (corto/largo).
2. Evaluar ticket medio.
3. Evaluar las capacidades internas del cliente antes de recomendar canales complejos (tiempo, habilidades, equipo, disposición a grabar vídeo, capacidad de escribir, presupuesto y herramientas disponibles).
4. Descartar canales donde el cliente objetivo no busca la solución o no hay capacidad interna para sostenerlo.
5. Seleccionar 1 canal principal de captación.
6. Seleccionar 1 canal secundario de retención/confianza.
7. Justificar la elección de cada canal.

## Salida esperada
`outputs/plan_actual/06_matriz_canales_marketing.md`

## Gates relacionados
`gate_canales_justificados`

## Criterios de insuficiencia
Bloquea si sugiere canales masivos (ej. TikTok) para negocios B2B hiperespecíficos sin justificación sólida.

## Errores frecuentes
- Recomendar "estar en todas las redes sociales".
- Sugerir canales incompatibles con los recursos o el tipo de venta.

## Límites
No decide presupuesto exacto, solo prioriza los canales.
