# Reglas del Sistema (Rules)

Aquí se definen las restricciones y normas de comportamiento inamovibles del sistema. Estas reglas aseguran que el Plan de Marketing se mantenga dentro de los límites éticos, técnicos y comerciales definidos.

## Ejemplos de reglas previstas:
- **No regenerar sin autorización**: Las fases que han sido ajustadas manualmente por un experto no pueden ser sobrescritas por el pipeline automatizado sin aprobación explícita.
- **No prometer sin datos**: Queda prohibido afirmar que una estrategia ha sido "validada comercialmente" si no existen registros reales en la Bitácora de Objeciones o en el Registro de Seguimiento.
- **Uso restringido de herramientas**: No se permite el uso de internet, MCP o APIs externas para la generación de contenido estratégico salvo autorización específica del usuario.
- **Prudencia estratégica**: El tono de los documentos debe ser profesional y realista, evitando promesas de resultados garantizados.
