# Gates de Validación

Los *Gates* son puntos de control obligatorio que bloquean el avance del sistema si no se cumplen criterios mínimos de calidad o coherencia.

## Gates previstos:
- **Brief Validado**: El proyecto no puede avanzar a la Fase 02 si los campos obligatorios no están completos y verificados.
- **Fase Previa Aprobada**: No se puede generar una fase (ej. Canales) si la fase anterior (ej. Competencia) no está marcada como completada.
- **Revisión de Sesgos**: Control para asegurar que las recomendaciones no son genéricas y se adaptan al perfil de marketing detectado.
- **Revisión de Prudencia**: Validación de que el lenguaje utilizado en los documentos de salida no exagera capacidades ni resultados.
