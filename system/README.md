# System

Esta carpeta contiene la capa de control operativo del sistema agéntico. Aquí residen las reglas, flujos de trabajo (workflows) y puntos de control (gates) que rigen el comportamiento del sistema.

**Nota importante**: Los archivos en esta carpeta son descriptivos y documentales. No contienen código ejecutable en Python, sino definiciones que los agentes (humanos o IA) deben seguir para garantizar la coherencia del Plan de Marketing.

## Estructura
- `rules/`: Reglas de comportamiento y restricciones.
- `workflows/`: Pasos operativos para tareas comunes.
- `gates/`: Criterios de validación obligatorios.
