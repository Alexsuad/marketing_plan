# Workflows Operativos

Esta sección documenta los pasos secuenciales necesarios para realizar tareas complejas dentro del sistema de forma repetible y segura.

## Workflows principales previstos:
1. **Creación de Proyecto**: Pasos desde la ejecución del comando hasta el llenado del brief inicial.
2. **Validación de Brief**: Proceso de revisión manual y técnica para asegurar que el negocio está bien definido antes de empezar.
3. **Generación de Fases**: Secuencia lógica de ejecución del pipeline de marketing.
4. **Revisión de Fase**: Protocolo para revisar un documento generado, aplicar ajustes y marcarlo como listo.
5. **Cierre de Hito**: Pasos para documentar el fin de un bloque de trabajo (ej. Días 1-30) y preparar el siguiente.
