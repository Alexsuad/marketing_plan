# Agentes del Sistema

Esta carpeta contiene la definición de roles y responsabilidades de las entidades agénticas que operan en el sistema.

**Nota**: Estos archivos son documentales (definición de rol). No son scripts ejecutables. Sirven para que el operador (humano o IA) sepa cómo actuar bajo ese rol específico.

## Agentes definidos:
- `orquestador_plan_marketing`: Coordina el sistema completo de generación del Plan de Marketing.
- `investigador_marketing`: Analiza información del negocio, mercado, competencia, contexto y señales disponibles.
- `estratega_marketing`: Convierte investigación y contexto en decisiones estratégicas de marketing.
- `redactor_marketing`: Convierte decisiones estratégicas en documentos claros, coherentes y utilizables.
- `analista_metricas`: Define el sistema básico de medición del Plan de Marketing.
- `auditor_plan_marketing`: Revisa el Plan de Marketing completo o una fase específica antes de avanzar o cerrar.
