# Skills (Habilidades)

Esta carpeta contiene instrucciones detalladas y reutilizables para realizar tareas técnicas o estratégicas específicas dentro del proceso de marketing.

Las skills permiten que el conocimiento obtenido en un proyecto se estandarice y pueda ser aplicado por cualquier agente en el futuro.

## Categorías
- `marketing/`: Habilidades relacionadas con el análisis, estrategia y redacción de marketing.
