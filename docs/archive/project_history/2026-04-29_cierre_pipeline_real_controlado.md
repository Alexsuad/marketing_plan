## Aprendizajes principales
- **Robustez Técnica:** El pipeline de marketing automatizado funciona como estructura base sólida.
- **Necesidad de Supervisión:** El brief inicial es un buen arranque, pero la estrategia de alto nivel requiere supervisión humana para evitar "respuestas promedio".
- **Contextualización:** La segmentación y el lenguaje ganan potencia cuando se adaptan al "dolor" real del negocio.
- **Prudencia:** Es vital que el lenguaje de marketing sea veraz y no declare validaciones comerciales antes de tener evidencia real.
- **Naturaleza del Sistema:** El sistema es una herramienta de asistencia y aceleración, no un generador de "caja negra" desatendido.
